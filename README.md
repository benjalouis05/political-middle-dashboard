# The political middle: humans vs language models

Interactive dashboard comparing how 18,637 General Social Survey respondents
(2014-2024) and four open-weight language models represent political moderates —
a test of the Fowler et al. (2023) vs Broockman & Lauderdale (2025) "Moderates"
debate, run inside LLMs and against human microdata on the same 14 items.

**Live page:** https://benjalouis05.github.io/political-middle-dashboard/

Built from the `llm-liar` research codebase (`dashboard/build_data.py`); static,
dependency-free (vanilla JS + inline SVG), light/dark mode automatic.
